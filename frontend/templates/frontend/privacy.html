Privacy Policy

What does this page store?

The following statements are true:

  • This page stores logs of its page-views.
  • These logs are deleted regularily.
  • In case of a malfunction, the developer of this page may choose to view the logs to find the problem. Log retention may be prolonged in this case.
  • These logs serve an integral purpose in the maintenance of the webpage. (Article 6, Paragraph 1, Point F, GDPR)
  • Our page, by itself, stores a single cookie. An authentication token to ensure the security of this page.
    This page will not work properly if this cookie is not set.

What other services are involved while using this page?

The following statements are true (to the best of our knowledge):

  • This service provides code to download your chosen software. You can review this code by clicking the download link of the release.
  • During install of the software provided by this page, the software is downloaded by third-party servers.
  • Arbitrary code may be executed by the software downloaded through this service.
  • Most downloads go through GitHub. Please refer to this page for their privacy policy.
  • We are using CloudFlare.
    • We are using CloudFlare. CloudFlare uses cookies to identify you. Refer to their privacy policy.
    • CloudFlare uses cookies to authenticate you so you won't be shown any captchas on your visit.
    • CloudFlare injects code that detects your browser version and injects additional context if you are using an outdated browser.

Can I view what you stored about me?

We don't collect anything about you. All information that could identify you resides on CloudFlare servers. Please refer to them.
